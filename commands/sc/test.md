# /sc:test - Testing and Quality Assurance

## Purpose

Execute tests, generate comprehensive test reports, and maintain test coverage standards.

## Usage
```
/sc:test [target] [--type unit|integration|e2e|all] [--coverage] [--watch] [--fix]
```

## Execution

1. Discover and execute tests with appropriate configuration
2. Monitor results, collect metrics, generate reports
3. Provide recommendations for test improvements
