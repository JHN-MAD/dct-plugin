# /sc:explain - Code and Concept Explanation

## Purpose

Deliver clear, comprehensive explanations of code functionality, concepts, or system behavior.

## Usage
```
/sc:explain [target] [--level basic|intermediate|advanced] [--format text|diagram|examples]
```

## Execution

1. Analyze target code or concept, identify key components and relationships
2. Structure explanation based on complexity level with examples
3. Present clear, accessible explanation with proper formatting
