# COMMANDS.md - SuperClaude Lite Commands

## Command Format
```
/sc:[command] [args] [--flags]
```

## Available Commands (8)

### Development
- `/sc:build [target]` — Build/compile/package with framework detection
- `/sc:implement [feature] [--type component|api|service|feature]` — Feature implementation with persona activation

### Analysis
- `/sc:analyze [target] [--focus quality|security|performance|architecture]` — Multi-dimensional code analysis
- `/sc:troubleshoot [issue] [--type bug|build|performance]` — Problem investigation and resolution
- `/sc:explain [target] [--level basic|intermediate|advanced]` — Educational explanations

### Quality
- `/sc:improve [target] [--type quality|performance|maintainability]` — Evidence-based code enhancement
- `/sc:test [target] [--type unit|integration|e2e|all] [--coverage]` — Test execution and reports

### Version Control
- `/sc:git [operation] [--smart-commit]` — Git with intelligent commits
